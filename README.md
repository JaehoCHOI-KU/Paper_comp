# 목포해양대 연구실적 비교 모바일 앱

대학정보공시(대학알리미)에 공개된 최근 3개 공시연도 연구실적을 스마트폰에서 비교하기 위한 정적 웹앱입니다.

## 비교 대상

- 국립목포해양대학교
- 국립한국해양대학교
- 국립군산대학교
- 국립목포대학교
- 국립순천대학교

## 비교 지표

- SCI급 논문
- 등재·후보지 논문
- 저역서
- 전체 연구성과 합계

## 데이터 교체 방법

현재 포함된 `data/research-data.csv` 값은 앱 동작 확인용 기본 데이터입니다. GitHub 공유 전 대학알리미에서 동일 항목의 최근 3개 공시연도 값을 확인한 뒤 아래 형식으로 교체하세요.

```csv
year,university,sci,registered,books
2026,국립목포해양대학교,0,0,0
```

앱 화면의 `데이터` 탭에서도 CSV를 불러와 즉시 확인할 수 있습니다.

## GitHub Pages 배포

1. 이 폴더의 파일을 GitHub 저장소에 업로드합니다.
2. 저장소 `Settings` > `Pages`로 이동합니다.
3. `Source`를 `Deploy from a branch`로 선택합니다.
4. Branch는 `main`, 폴더는 `/root`로 지정합니다.
5. 저장 후 배포 주소를 확인합니다.

서버 빌드가 필요 없는 앱이라 GitHub Actions보다 branch 배포 방식이 안정적입니다.
